Show that the ACF of \({X_t }\) is zero except at lag 0; \[ \delta(h) = \frac{\gamma(h)}{\gamma(0)}, \text{where} \gamma(h), \gamma(0) \text{ are auto covariance and varaince}\\ E(X_t) = \frac{0.8 E(\epsilon^2_{t-1})}{1+E(\varepsilon^2_{t-1})} + E(\varepsilon_{t})\\ = \frac{0.8\sigma^2}{1+\sigma^2}+0\\ \gamma(0) = Var(X_t) = \frac{0.8 Var (\varepsilon^2_{t-1})}{1+Var(\varepsilon^2_{t-1})} + \sigma^2\\ E(\varepsilon^2_{t-1} )= Var(\varepsilon_{t-1})+[E(\varepsilon_{t-1})]^2 = \sigma^2\\ cov(X_t,X_{t+1}) = cov(\varepsilon _ { t } + \frac { 0.8 \varepsilon _ { t - 1 } ^ { 2 } } { 1 + \varepsilon _ { t - 1 } ^ { 2 } },\varepsilon _ { t+1 } + \frac { 0.8 \varepsilon _ { t } ^ { 2 } } { 1 + \varepsilon _ { t } ^ { 2 } }) = 0.8 cov(\epsilon_t, \frac { \varepsilon _ { t - 1 } ^ 2}{1+\varepsilon _ { t - 1 }}) = 0\\ \text{similarly we have} cov(X_t,X_{t+2}) = 0\\ \text{ACF is: 1 if h=0, else: ACF=0} \]

(c). Use (b) to show that the PACF of \({X_t }\) is zero. For PACF: \[ \sigma_{kk} = \frac{f_k-\sum_{j=1}^{k-1}\alpha_{k-1,j}f(k-j)}{1-\sum_{j=1}^{k-1}\alpha_{k-1,j}f(j)}\\ \sigma_{kj} = \alpha_{k-1,j}-\alpha_{kk},\alpha_{k-1,j-1}\\ \alpha_{11} = \alpha_{22} = \alpha_{12} = 0\\ \] So we have PACF is zero.

This example shows that ACF and PACF are useful mainly for linear time series.
